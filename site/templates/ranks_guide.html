Гайд по системе рангов


Начало

Ранговая система предназначена для определения лучших игроков на каждом режиме игры. В течении дня система VimeArchive собирает статистику всех игроков и считает их очки ранга.
По количеству очков создается топ на каждый режим, за достижение определенных мест в топе, игрокам выдается значек.

Формулы

Система рангов стремится определить наиболее скиловых игроков, по этому для каждого режима очки считаются по своему. Ниже предоставлены формулы для каждого режима.
Режим Формула
Annihilation kills+bowkills*2+nexus*2
BuildBattle wins*(wins/games)
BlockParty wins*(wins/games)
The Bridge (wins*2+points)*(wins/games)+kills*(kills/deaths)
LuckyWars wins*2*(wins/games)+kills*(kills/deaths)
BedWars (wins+bedBreaked)*2*(wins/games)+kills*(kills/deaths)
BedWars Turbo (wins+bedBreaked)*2*(wins/games)+kills*(kills/deaths)
ClashPoint (wins+resourcePointsBreaked)*2*(wins/games)+kills*(kills/deaths)
DeathRun wins*(wins/games)
Duels solo_wins*(solo_wins/solo_games)+team_wins*(team_wins/team_games)+ranked_wins*(ranked_wins/ranked_games)
GunGame (wins*3+kills)*(wins/games)
HungerGames (wins*3+kills)*(wins/games)
KitPvP (points+kills)*(kills/deaths)
MobWars wins*(wins/games)
Prison kills*(kills/deaths)+(total_blocks/10)+(mobs/2)
SkyWars wins*2*(wins/games)+kills*(kills/deaths)
Аркады wins*2*(wins/games)+kills*(kills/deaths)
JumpLeague (wins*2+checkpoints)*(wins/games)+kills*(kills/deaths)
Murder Mystery (wins_as_innocent_p+wins_as_maniac_p+wins_as_detective_p)/(total_wins/games)+kills
SheepWars wins*2*(wins/games)+kills*(kills/deaths)
Turf Wars wins*2*(wins/games)+kills*(kills/deaths)
TNT Tag (wins*2+kills)*(wins/games)
Царство Зомби Мороза (wins*2+kills)*(wins/games)
Hide and Seek (wins*2+kills)*(wins/games)
Speed Builders wins*(wins/games)
Fall Guys wins*(wins/games)
Team Fortress wins*2*(wins/games)+kills*(kills/deaths)
Egg Wars wins*2*(wins/games)+kills*(kills/deaths)

Возможно для некоторых режимов можно составить более грамотную формулу и если вы это сделали, то напишите в дискорд: Fa#0287
